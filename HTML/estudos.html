Tipos de Tags


Títulos:

Podemos usar 6 tipos de tagas para títulos.

Essas tags são representadas pela a letra H e por números de 1 ao 6

Exemplo: < h1 >...< h6 >

Parágrafos:

Os parágrafos são representados pela tag < p >

Negrito e Itálico:

Para destacar uma palavra em negrito, usamos a tag < b >

Para destacar uma palavra em itálico, usamos a tag < i >

Sobrescrito e Subescrito:

Para deixar uma letra ou palavra superior as demais,usamos a tag < sup >
Exemplo: EMC2

E para sobescrever uma palavra ou letra usamos a tag < sub >
Exemplo: H2O


Quebra de linha:

Para quebra de linha usamos a tag < br >

Linhas Horizontais:

Para traçar uma linha horizontal use a tag < hr >

Marcações Semânticas

Forte e Ênfase

Para uma marcação de grande relevância usamos a tag Strong, que é representada pela tag < strong > .
O que estiver dentro dessa tag, vai está representada em negrito.

Um outra tag para se dar ênfase a uma determinada frase em um texto é a tag < em >
O trecho que estiver dentro dessa tag vai estar em ítalico.

A próxima tag desse segmento será atag blockquote < blockquote >
Essa tag faz com que o que estiver dentro dela , tenha um recuo de destaque dentro de outro texto como no exemplo.
É usada em citações maiores dentro de um texto.

Tudo posso naquele que me fortalece

JESUS CRISTO VIVE!

O blockquote nãe é uma tag muito utilizada

A última tag desse tópico é a tag < q >
Essa tag destaca entre aspas o que estiver dentro dela, e é usada em citações menores

Por que DEUS amou o mundo de tal maneira,que deu o seu filho primogênito para que todos que nele crê
não pereça, mais tenha vida eterna .

Abreviações e Acrônimos

Podemos usar abreviaturas e acrônimos em um texto e uma tag que pode ajudar é a tag < abbr >
Dentro dessa tag colocamos os significado da abreviatura ou acrônimo.
Exemplo:

Ao Sr. Fulano,desejamos um feliz natal!

Citações e Definições

A tag a seguir é usada para citação de uma fonte da qual você citou o texto ou mesmo o nome do autor. tag < cite >
Exemplo:

O Senhor do Anéis é um dos melhores contos de J.R.R Talkien.

A tag a seguir é usada para uma definição de um jargão ou mesmo um conceito. Ela é representada pela tag < dfn >
Ela é reprentada em ítalico menos em navegadores Chrome e Safari

Detalhe do Autor

Com essa tag podemos direcionar o usuário para mais detalhes,fazendo uso de link para uma página especíica
A tag usada é: < address >

Exemplo:

Mais sobre Stephen Hawking Cientista da área de teoria quântica
Perceba que uma tag de link foi usada para direcionar o usuário para outra página.

Alterações em conteúdo

Para alterar um conteúdo como exemplo uma promoção.Um valor X para um valor Y e vamos riscar o valor antiga
Para isso usaremos a tag < del > . Essa tag vai representar que um elemento foi excluido e a tag
< ins > identifica que um novo elemento foi adicionado
Exemplo:

PROMOÇÃO: Havaianas de:R$ 19,90 Por :R$16,90

Para uma ocasião em que um elemento nãp está mais correto usamos a tag < s >
Essa tag aparece da mesma forma que a tag < del > mais são interpretadas diferente pelo o desenvolvedor.
Exemplo:
De: R$ 10,90 por: R$ 9,90
